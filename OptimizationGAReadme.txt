"OptimizationGA.py"

Description:
The file for solving the optimization problem with the Genetic algorithm

Guidelines:
- Comment out one of the grids named grid1 as you want to run the optimization for (row 380-422),
- or comment out gridOrg at row 377 to generate random grid and comment in gridOrg at row 424

Set corresponding values:
- gridsize at row 373
- obstacles at row 374
- numberOfSwords (numberOfMonsters) at row 12

Run from terminal: "python3 OptimizationGA.py"
